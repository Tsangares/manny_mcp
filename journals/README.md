# Journal Organization

This directory contains development journals documenting lessons learned during manny plugin development and OSRS automation.

## Directory Structure

- **combat/** (4 files) - Combat system issues, targeting, interference, training
- **navigation/** (4 files) - Indoor navigation, door handling, ferry travel, transitions
- **plugin-dev/** (8 files) - Plugin commands, argument parsing, GE development, session recording
- **quests/** (5 files) - Quest automation, dialogue handling, Tutorial Island
- **skilling/** (5 files) - Fishing, cooking routines, timing analysis
- **system/** (4 files) - Threading issues, crashes, process management
- **ui-widgets/** (8 files) - Deposit box, shop widgets, inventory clicking, widget inspection

## Usage

Journals document **lessons for future agents**, not activity logs. Each entry should focus on:
- Root cause analysis with code references
- BAD vs GOOD patterns with examples
- Debugging techniques that worked
- Interface boundaries and gaps

See `TEMPLATE.md` for journal structure and guidelines.
